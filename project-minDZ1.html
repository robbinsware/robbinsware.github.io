MinDZ Calculator

Description

  Excel based project that helps C-17A airdrop crews calculate the minimum requiremnts for a Drop Zone.

Initial Deployment

  Initially launched in 2011.
  Continually updated and always under review to ensure compliance with current AFIs.

Supported Platforms

  Requires Macro-Enabled Microsoft Excel.

History

  Idenfitying the need for a more efficient and accurate way to navigate the regulations that specify the legal requirements for USAF C-17 Airdrop operations, the project was developed to leverage software to aid crews in mission planning. Previous projects like "Bakeware" met the need for years but had not recently been updated and also did not provide a visual representation.
